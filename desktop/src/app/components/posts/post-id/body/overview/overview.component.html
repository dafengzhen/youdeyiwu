描述露营中的知识、野炊等户外活动的魅力和以此作为娱乐的女高中生的温和日常生活。漫画标题末端的“△”是代表露营帐篷的象形符号。于《Manga Time Kirara Forward》的2015年的7月号连载至2019年4月号[1]，自2019年春季起移籍至《COMIC FUZ》继续连载[2]。台湾中文版由东立出版社代理，中国大陆中文版电子书由哔哩哔哩漫画代理、实体书由天闻角川代理。2016年4月9日起，幕间故事的番外篇《房间露营△》（日语：へやキャン△）于Niconico静画的Kirara基地以每话两页的形式连载中。 2019年11月20日发表制作电视剧消息，于2020年1月至3月播出[3]。2020年11月5日发表制作电视剧第二季，2021年4月开始播送[4]。
